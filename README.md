# ngrx-learning-seed

### Dependencies
    - [Angular-CLI](https://cli.angular.io/)
    
### Instructions for Cold Start 
    ```sh
    $ ng new demoapp
    $ cd demoapp
    $ npm install @ngrx/store --save 
    ```
### Instructions for Cloned Start
    ```sh
    $ git clone ...
    $ cd demoapp
    $ ng serve
    ```
